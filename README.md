# warming-age-coinfection
Metadata and R code for Higher Temperature and Host Age Alter Infection Outcomes in a Multi-Pathogen System

# metadata column titles
#replicate: temporal replicate (1:6)

#temp: temperature treatment (21/29)

#age_gp: y (young, 7 week) or o (old, 13 week)

#inoc: inoculum (c = C. cereale only, r = R. solani only, rc = co-inoculation, m = mock)

#X_D5-D20: infection presence (1) or absence (0) of that infection type on that day (ex. D5 = survey Day 5)

#pla_DX: percent leaf area affected on that day

#tillersDX: number of tillers on that day

#dry.biomass: dry aboveground biomass of plants which survived to day 20

#plant_ID: unique identifier
